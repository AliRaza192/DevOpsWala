# PHASE 13: AGENTIC AI — Agent Building
*Est. Time: 6-8 weeks*

### Kya seekhna hai
- Agentic Coding (Claude Code, OpenCode CLI workflows)
- AI Agent frameworks comparison:
  - **OpenAI Agents SDK** — elegant, flexible
  - **LangChain + LangGraph** — graph-based, stateful, production-grade
  - **CrewAI** — low-code, fast prototyping
  - **AutoGen (Microsoft)** — multi-agent remote collaboration
  - **Claude Agent SDK / Claude Code SDK** — Anthropic native
  - **Google ADK (Agent Development Kit)** — Google native
  - **PydanticAI** — type-safe agents
- **MCP (Model Context Protocol)** — tools/data ko agents se connect karna
- Tool calling / function calling
- Agent vs Workflow pattern

### Frameworks Comparison
| Framework | Best For | Difficulty |
|-----------|----------|------------|
| **LangGraph** | Complex stateful agents, Production | Advanced |
| **CrewAI** | Multi-agent teams, Fast prototyping | Beginner |
| **AutoGen** | Conversational multi-agent | Intermediate |
| **OpenAI Agents SDK** | OpenAI-native, Elegant | Intermediate |
| **Claude Agent SDK** | Anthropic-native, MCP | Intermediate |
| **Google ADK** | Google-native, Multi-agent | Intermediate |
| **PydanticAI** | Type-safe, Python-native | Beginner |
