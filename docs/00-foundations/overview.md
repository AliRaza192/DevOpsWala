# PHASE 0: Computer & Mindset Foundations
*Jis ko bilkul computer ki basic knowledge ho, wo bhi yahan se start kar sakta hai*

### Kya seekhna hai
- Computer kaise kaam karta hai (CPU, RAM, Storage, OS)
- Internet kaise kaam karta hai (client-server model)
- File aur folder ka concept, file extensions
- Keyboard shortcuts, typing speed
- "AI Era Mindset" — AI agent ko director ki tarah use karna, sirf coder ki tarah nahi

### Hands-on Checklist
- [ ] Computer on/off, files create karo
- [ ] Command line open karke basic commands chalao (cd, ls, dir, ping)
- [ ] Internet search karo aur samjho IP address kya hota hai

### Meri Recommendation
Tumhe ye phase skip level pe hai (already Linux/AI daily use karte ho) — lekin AgentFactory ka "How to Think in the AI Era" aur "AI Prompting in 2026" zaroor parho.
