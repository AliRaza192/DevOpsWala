# PHASE 4: Cloud Fundamentals — AWS + Azure + GCP
*Est. Time: 6-8 weeks | AWS primary + Azure (Gulf market) + GCP awareness*

### Kyun teeno clouds?
- **AWS** — globally sabse zyada market share, US/Europe/Remote jobs k liye primary
- **Azure** — Gulf (UAE, Saudi) aur enterprise companies mein dominant
- **GCP** — kam demand lekin AI/ML-heavy companies mein use hota hai

### AWS — Core Concepts
- **IAM** — users, roles, policies, least-privilege
- **EC2** — virtual servers, AMIs, auto-scaling
- **S3** — object storage, lifecycle policies
- **VPC** — subnets, route tables, NAT, security groups
- **RDS** — managed databases
- **Lambda** — serverless compute
- **EKS** — managed Kubernetes
- **CloudWatch** — monitoring/logging
- **Route 53** — DNS

### Azure — Gulf Market k Liye Zaroori
- VMs, Storage Accounts
- VNets, NSGs
- **AKS** (Azure Kubernetes Service)
- **Azure DevOps** — pipelines, repos, boards
- Azure IAM (RBAC), Azure Cost Management

### GCP — Awareness
- Compute Engine, Cloud Storage, VPCs/Firewalls

### Hands-on Checklist
- [ ] VPC with public/private subnets, NAT gateway
- [ ] IAM least-privilege role setup
- [ ] EC2/VM deploy with security group
- [ ] S3/Blob lifecycle policy
- [ ] Lambda function banao jo S3 event trigger kare

### Incident Practice
- Access/network issue fix karo (NSG/IAM role)
- Load balancer/ingress traffic routing issue fix karo
- Cost-aware tagging set karo

### Hybrid Architecture Model
Poora project migrate mat karo. 2026 pattern: **Frontend + API layer → Vercel/SST**, **Heavy workloads (agent harness, K8s) → AWS EKS / Azure AKS**.

### Meri Recommendation
Order: AWS Cloud Practitioner concepts → Azure AZ-900 (Gulf jobs must) → AWS EKS/IAM deep dive → GCP awareness only.
